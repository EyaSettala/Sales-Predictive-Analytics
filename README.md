# 📊 Sales Predictive Analytics

An end-to-end Business Intelligence and Machine Learning solution for sales analysis and forecasting. This project centralizes sales data, provides interactive dashboards, predicts future revenue, and helps decision-makers identify trends through a user-friendly web platform.

## 🚀 Features

* 🔄 ETL pipeline for extracting, transforming, and loading sales data
* 🗄️ Data warehouse for centralized and structured storage
* 📈 Interactive dashboards with key performance indicators (KPIs)
* 🤖 Machine Learning models for sales and invoice forecasting
* 📊 Trend analysis to support strategic decision-making
* 🌐 Modern web application with authentication and profile management
* 🖨️ Export and visualization capabilities for reports and predictions
